## 스택
* 스택은 한쪽 끝에서만 자료를 넣고 뺄 수 있는 자료구조이다.
* 값을 넣을떄 push
* 값을 제거할떄 pop
* 맨 위의 값은? - top
### 스택의 구현
* 일차원 배열 하나로 구현할 수 있다.
~~~~~~
int stack[10000];
int size = 0; -- 스택의크기
~~~~~~

#### push 구현
~~~~~~
void push(int data){
  stack[size] = data;
  size += 1;
}
# 사이즈 크기만큼의 인덱스 위치에 값을 넣고, 사이즈의 값을 증가시켜주는 식으로 구현(push)
~~~~~~

#### pop 구현
~~~~~~
int pop(){
  stack[size-1] = 0;
  size -= 1;
 }
~~~~~~


* python은 list는 stack 역할을 하기때문에 따로 구현하지 않아도된다.

### LIFO
* Last In First Out

괄호문제,단어바꾸기문제,스택수열, -- 스택활용의 예.  

### 스택수열
* 1부터 N가지의 수를 스택에 넣었다가 뽑아 놓음으로 하나의 수열을 만들 수 있다.
* push하는 순서는 오름차순이다.
* 임의의 수열 A가 있을 때, 스택을 이용해 이 수열을 만들 수 있는지 없는지 구하는 문제.
* 스택에 push되는 순서는 오름차순이다.
* pop되는 순서대로 수열 A가 만들어지기 때문에, A의 첫 수 부터 순서대로 만ㄷ르어보면 된다.
